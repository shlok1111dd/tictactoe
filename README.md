# Tic‑Tac‑Toe (Web)

Simple single-file Tic‑Tac‑Toe you can open in a browser. Controls:
- Click a cell to place `X` (human). AI plays `O` using minimax.
- Use "AI goes first" to let the AI start.

Run locally:

Open `index.html` directly in your browser, or run the simple Node server below.

PowerShell / cmd:

```powershell
# start Node static server
npm install   # optional, no deps required
npm start

# then open http://localhost:3000
```

Or open the file directly:

```powershell
start index.html
```
